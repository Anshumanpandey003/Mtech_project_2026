# MTech Project 2026 - Grievance Redressal System

> A React-based Grievance Redressal System built as part of MTech project at IIT Jodhpur (2026)

## Live Demo

[View Live App](https://anshumanpandey003.github.io/Mtech_project_2026)

## About

This project is a **Grievance Redressal System** developed using React.js. It allows users to submit, track, and manage grievances through a clean and user-friendly interface.

## Features

- Submit grievances with category, priority, and description
- Track grievance status in real-time
- Admin dashboard to manage and resolve grievances
- Responsive UI built with React

## Tech Stack

- **Frontend:** React.js, JSX, CSS
- **Deployment:** GitHub Pages / Vercel
- **Version Control:** GitHub

## Getting Started

### Prerequisites
- Node.js (v14+)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Anshumanpandey003/Mtech_project_2026.git

# Navigate to project folder
cd Mtech_project_2026

# Install dependencies
npm install

# Start development server
npm start
```

### Build for Production

```bash
npm run build
```

### Deploy to GitHub Pages

```bash
npm run deploy
```
